Sound Credits
============

This directory contains sound effects for the Webora website:

1. switch-click.mp3 - A light switch click sound effect for the dark/light mode toggle

You can obtain this sound in one of these ways:

Option 1: Download a free light switch sound from a royalty-free sound library:
- Freesound.org: https://freesound.org/search/?q=light+switch
- Pixabay Sounds: https://pixabay.com/sound-effects/search/light%20switch/
- Zapsplat: https://www.zapsplat.com/sound-effect-categories/switches/

Option 2: Record a sound of your own light switch clicking

Option 3: Use an online audio converter to convert from another format to MP3 if needed

Make sure to rename the downloaded file to "switch-click.mp3" and place it in this directory.
